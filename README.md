# Toolbar Template in Gantt Chart

This repository contains a sample Blazor Web Application that demonstrates how to customize the toolbar of the Syncfusion Blazor Gantt Chart using a toolbar template. The sample focuses on modifying the appearance and layout of toolbar items while preserving the built‑in Gantt Chart functionality and behavior.

## Project overview

The Syncfusion Blazor Gantt Chart provides a built-in toolbar with commonly used actions such as add, edit, delete, expand, and collapse. This project illustrates how the toolbar can be customized using a template to achieve tailored layouts, custom styling, and additional toolbar content.

The toolbar template enables flexible rendering of toolbar items, that allows to align the Gantt Chart toolbar with application‑specific UI requirements and branding guidelines. The sample demonstrates how the customized toolbar integrates seamlessly with the Gantt Chart component while ensuring that standard toolbar actions continue to function as expected.

This approach is useful when the default toolbar layout needs visual enhancement or extended customization without altering core Gantt Chart functionality.

## Features

- Custom toolbar rendering using toolbar templates
- Seamless integration with Syncfusion Blazor Gantt Chart
- Retention of built-in toolbar actions
- Flexible layout and styling for toolbar content
- Implementation using Syncfusion Blazor Gantt in a Blazor Web Application

## Prerequisites

- Visual Studio 2022 (or later)
- .NET SDK 8.0 or later
- Syncfusion Blazor Gantt NuGet package
- A valid Syncfusion license (Community or Trial)

## How to run the project

1. Clone or checkout this repository to a local folder.
2. Open the project file (`.csproj`) in Visual Studio 2022 or later.
3. Restore the required NuGet packages.
4. Run the application.
5. Navigate to the page containing the Gantt Chart.
5. Review the customized toolbar rendered using the toolbar template.
