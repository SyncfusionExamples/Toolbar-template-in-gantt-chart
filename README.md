# Toolbar Template in Gantt Chart

This repository contains a sample application that demonstrates how to render the Syncfusion Blazor Gantt Chart with a customized toolbar using a toolbar template. The sample focuses on modifying the appearance and layout of toolbar items while retaining the default Gantt Chart functionality.

## Project Overview

The Syncfusion Blazor Gantt Chart includes a built-in toolbar that provides commonly used actions such as add, edit, delete, expand, and collapse. This project illustrates how the toolbar can be customized using a template to achieve tailored layouts and custom toolbar content. The toolbar template allows flexible rendering of toolbar items to match specific application design requirements.

The sample highlights how a toolbar template integrates seamlessly with the Gantt Chart component and how standard toolbar actions continue to function as expected. This approach is useful in scenarios where the default toolbar layout needs visual customization or extended styling.

## Features

- Custom toolbar rendering using toolbar templates
- Integration with Syncfusion Blazor Gantt Chart
- Retention of built-in toolbar actions
- Flexible layout and styling for toolbar content
- Implementation in a Blazor application

## Prerequisites

- Visual Studio 2026
- .NET SDK compatible with Blazor
- Syncfusion Blazor Gantt NuGet package

## How to Run the Project

1. Clone or checkout this repository to a local folder.
2. Open the solution file in Visual Studio 2026.
3. Rebuild the solution to restore the required NuGet packages.
4. Run the application and navigate to the page containing the Gantt Chart.
5. Review the customized toolbar rendered using the toolbar template.
